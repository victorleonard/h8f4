## Analyse par site

### 1. Skynyrd Reloaded — Référence cover/tribute band mature

**Points forts**
- Hero avec dates à venir immédiatement visibles → crédibilité « groupe actif »
- Bio courte + témoignages presse intégrés sur la homepage
- Vidéo live en section centrale
- Contact booking direct (téléphone + email) répété
- EPK dédié (photos presse, audio, vidéo)

**Points faibles pour notre cas**
- Site multi-pages (Shows, Band, EPK, Media…) — trop lourd pour notre contrainte 1–2 pages
- Longue liste de « past shows » qui alourdit le scroll mobile
- Design Bandzoogle générique, peu d'identité visuelle forte

**À retenir** : témoignages, vidéo live, CTA booking visible, preuve d'activité scénique.

---

### 2. Heavy Metal Band Template — Référence structure one-page

**Points forts**
- **One-page** avec ancres : bandinfo, members, shows, audio, video
- **Dark theme** (#121212) — ambiance rock immédiate
- Header fixe + menu hamburger mobile
- Minimal JS, performance excellente
- Sections claires, scroll fluide

**Points faibles**
- Esthétique metal très niche (pas adapté tel quel à un cover band multi-genres)
- Pas de formulaire de contact intégré

**À retenir** : architecture one-page par ancres, dark mode, mobile-first, simplicité technique.

---

### 3. Set (getset.cc) — Référence mobile & simplicité

**Points forts**
- Philosophie « un lien, tout dedans » — parfait pour partage Instagram/TikTok
- Design épuré, typo soignée, espacement généreux
- Lecteurs embed (Spotify, YouTube) sans friction
- Mobile-first explicite

**Points faibles**
- Trop minimal pour un groupe qui veut une galerie photo riche et un formulaire booking détaillé
- Peu de personnalité visuelle « rock »

**À retenir** : lisibilité mobile, hiérarchie claire, pas de surcharge navigation.

---

### 4. EPK Sites — Référence conversion booking

**Points forts**
- Site pensé **uniquement pour convaincre les bookers** (pas les fans)
- Formulaire booking intégré, mobile-first
- Light/dark mode
- Single-page EPK comme offre de base

**Points faibles**
- Esthétique très « corporate pro », manque d'énergie rock

**À retenir** : funnel de conversion vers le formulaire, champs booking structurés (date, lieu, type d'événement, budget).

---

### 5. Little Lies (Field Trip) — Référence direction créative tribute

**Points forts**
- Identité visuelle inspirée de l'univers du groupe hommagé, mais modernisée
- **Vidéo hero** en header — immersion immédiate dans l'énergie live
- Palette sophistiquée, typo cohérente
- Architecture orientée conversion venue booking

**Points faibles**
- Projet sur-mesure haut de gamme — budget et délai non comparables

**À retenir** : vidéo hero courte en boucle, identité visuelle forte, ton pro mais énergique.

### 8–9. Templates dark (Velvet, Acid, Entourage)

**Tendances visuelles communes**
- Fond sombre (#0D0D0D → #1A1A1A)
- Accent néon ou chaud (rouge, orange, or) pour CTA
- Typo display bold (titres) + sans-serif lisible (corps)
- Texture grain ou overlay léger sur les photos → ambiance « live / analogique »
- Animations subtiles au scroll (fade-in, pas de parallax lourd sur mobile)

## Synthèse des tendances

### Ambiance visuelle dominante

Les sites de groupes rock / cover efficaces partagent :

1. **Dark mode** — fond noir ou gris très foncé, texte clair
2. **Énergie live** — photo ou vidéo de scène en hero, pas de stock photo
3. **Contraste fort** — titres imposants, CTA coloré qui ressort (orange/rouge)
4. **Minimalisme fonctionnel** — peu de pages, navigation par ancres
5. **Crédibilité pro** — répertoire visible, contact facile, preuves sociales

### Layout type (one-page)

```
┌─────────────────────────────┐
│  Header fixe + menu burger  │
├─────────────────────────────┤
│  HERO                       │
│  Nom groupe + tagline       │
│  [Vidéo/photo live]         │
│  [CTA: Réserver le groupe]  │
├─────────────────────────────┤
│  À PROPOS                   │
│  Bio courte + photo groupe  │
├─────────────────────────────┤
│  RÉPERTOIRE                 │
│  Grille de titres           │
├─────────────────────────────┤
│  MÉDIAS                     │
│  Galerie photos + vidéos    │
├─────────────────────────────┤
│  CONTACT / BOOKING          │
│  Formulaire + tel/email     │
├─────────────────────────────┤
│  Footer — réseaux + légal   │
└─────────────────────────────┘
```

### Mobile

- Menu hamburger avec ancres
- CTA « Réserver » sticky en bas ou dans le header
- Vidéos en ratio 16:9, images en grille 2 colonnes max
- Formulaire en champs empilés, bouton submit pleine largeur
- Pas d'animations lourdes (respect `prefers-reduced-motion`)

## Direction créative recommandée

### Thème : **Dark Rock Energy**

| Élément | Choix recommandé |
|---|---|
| **Fond** | `#0A0A0A` (noir profond) ou `#121212` |
| **Surface / cartes** | `#1A1A1A` avec bordure subtile `#2A2A2A` |
| **Texte principal** | `#F5F5F5` |
| **Texte secondaire** | `#A0A0A0` |
| **Accent primaire** | `#E63946` (rouge rock) ou `#FF4500` (orange énergique) |
| **Accent CTA** | `#FF6B35` (orange chaud) — contraste fort sur fond sombre |
| **Typo titres** | **Russo One** ou **Bebas Neue** — impact rock |
| **Typo corps** | **Poppins** ou **Inter** — lisibilité mobile |
| **Effets** | Grain léger sur hero, hover glow subtil sur CTA, fade-in au scroll |
| **Photos** | Live, haute contraste, légère désaturation + overlay sombre |

### Ton éditorial

- Direct, énergique, professionnel
- Accroche type : *« Du rock iconique, live et sans compromis »*
- CTA : *« Réserver le groupe »* / *« Demander un devis »*

### Logo existant

Intégrer `assets/logo-no-background.svg` dans le header et le hero, en blanc ou couleur accent selon le fond.

## À privilégier

- One-page avec navigation par ancres
- Hero vidéo courte (3–5 s en boucle) ou photo live plein écran
- CTA booking visible dès le hero + répété avant le formulaire
- Répertoire en grille compacte (27 titres → 3 colonnes desktop, 2 mobile)
- Galerie photos masonry ou grille simple
- 1–3 vidéos YouTube embed (performances live)
- Formulaire booking : nom, email, téléphone, date, lieu, type d'événement, message
- Performance : images WebP, lazy loading, peu de JS

## À éviter

- Sites multi-pages avec navigation complexe
- Fond clair / esthétique « corporate »
- Trop d'animations (parallax, glitch, néon cyberpunk)
- Listes de concerts passés interminables
- Musique auto-play au chargement
- Formulaire caché ou contact uniquement par email
- Emojis comme icônes (préférer SVG : Lucide / Heroicons)

## Impact sur le choix framework

Le benchmark confirme la pertinence d'**Astro** :

| Besoin | Astro | Vue/Nuxt |
|---|---|---|
| One-page statique | ✅ Idéal | ⚠️ Surdimensionné |
| Galerie photos légère | ✅ | ✅ |
| Formulaire (service tiers) | ✅ | ✅ |
| Vidéos embed YouTube | ✅ | ✅ |
| Animations scroll subtiles | ✅ (CSS + îlots si besoin) | ✅ |
| Performance mobile | ✅✅ Excellent | ✅ Bon |
| SEO | ✅✅ | ✅ |

**Recommandation maintenue : Astro + Tailwind CSS**, avec îlot Vue optionnel uniquement si galerie interactive (lightbox, filtres).
